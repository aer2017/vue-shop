## 首页功能有 新闻资讯 图片分享 商品购买 
### 新闻资讯里的新闻详情最下面可以发表评论
### 进入图片分享 
+ 图片使用了mint-ui里面的懒加载组件
+ 上方的导航可以滑动 
+ 图片详情里面有使用 插件 vue-preview 做的缩略图,
+ 最下面可以发表评论
### 商品购买下方可以点击加载更多 商品详情里面可以实现加入购物车的功能 还有图文介绍

## 购物车
### 使用了vuex 实现数据的共享 可以实现商品数量的统计和总价计算
